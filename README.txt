# Virtual Pet Game

## Overview

A Python-based interactive virtual pet game where you can feed and play with your pet. The pet has hunger and happiness levels that change with your actions.

## Features

- Name your virtual pet.
- Feed and play with your pet to change its hunger and happiness levels.
- Check your pet's status.

## Technologies Used

- Python

## How to Run

1. Run `virtual_pet.py` using Python: `python virtual_pet.py`.

2. Follow the on-screen prompts to interact with your pet.
